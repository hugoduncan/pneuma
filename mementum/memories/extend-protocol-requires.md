Every pneuma.code.* extension file must `:require` the formalism/morphism namespace *before* the `:import` of its record class. The `:import` needs the defrecord class to already be compiled. The lean extensions got away without explicit requires because other require chains loaded the namespaces first, but pneuma.code.core loading the code extensions in isolation exposed ClassNotFoundException. Always pair `:require [pneuma.formalism.foo]` with `(:import [pneuma.formalism.foo FooRecord])`.
