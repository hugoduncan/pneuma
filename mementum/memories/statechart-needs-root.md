A statechart with no hierarchy has no roots, so `initial-config` returns an empty set and reachability analysis produces nil states. The lean emitter calls `name` on these nil values, causing NPE. Always provide at least a minimal hierarchy with a root: `{:hierarchy {:root #{:s1 :s2 ...}} :initial {:root :start-state}}`. This is also semantically correct — a flat statechart is really a single-level hierarchy.
