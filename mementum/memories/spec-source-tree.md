🎯 Formalism specifications live in spec/ source tree (not src/) with a -spec namespace suffix. E.g. pneuma.protocol's spec is spec/pneuma/protocol_spec.clj (ns pneuma.protocol-spec). This keeps specs out of the production jar — they're development/checking artifacts. The spec/ path is on :dev, :test, :lint, and :fmt classpaths in deps.edn. Convention generalizes: any namespace foo.bar gets its spec at spec/foo/bar_spec.clj (ns foo.bar-spec).
