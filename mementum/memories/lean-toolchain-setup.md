🎯 Lean 4 is installed via elan at ~/.elan/bin/lean (not on PATH by default). Use `PATH="$HOME/.elan/bin:$PATH" lake build` from proofs/. The proofs/ directory has its own lean-toolchain file pinning the version (currently v4.28.0). No Mathlib dependency yet — emissions are self-contained. For standalone file checks (e.g. in tests), run `~/.elan/bin/lean <file>` directly — no lake workspace needed since emissions have no imports. Lean outputs errors to stdout, not stderr.
